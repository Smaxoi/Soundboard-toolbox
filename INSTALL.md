# 安裝教學（一般使用者）

這份是給「**拿到安裝檔**」的人。如果你要從原始碼自己編譯，請看 [BUILD.md](BUILD.md)。

## 系統需求

- **Windows 10 / 11（64 位元）**
- 不需要額外安裝任何東西——程式是**自包**的（不必裝 Visual C++ 執行階段）。

---

## 一、安裝

1. **下載安裝檔**：[VoiceMod-Toolbox-Setup.exe](https://github.com/Smaxoi/Soundboard-toolbox/releases/latest/download/VoiceMod-Toolbox-Setup.exe)
   （或到 [Releases 頁面](https://github.com/Smaxoi/Soundboard-toolbox/releases) 選版本）。
2. **雙擊**它。
3. 若出現藍色的「**Windows 已保護你的電腦 (SmartScreen)**」視窗 → 點「**其他資訊**」→「**仍要執行**」。
   > 這是因為程式**未經數位簽章**（不是病毒），個人軟體很常見。
4. 照精靈一直按 **Next → Install**。**不需要系統管理員權限**（裝在你的使用者資料夾）。
5. 完成後可勾選「Launch」直接開啟，或從**開始功能表／桌面**的「**VoiceMod Toolbox**」捷徑開啟。

> 某些防毒可能對「全域熱鍵」功能跳提示（程式用鍵盤掛鉤偵測你設定的熱鍵，**不會記錄你打的字**）。若被攔截，把它加入信任即可。

---

## 二、第一次使用

開啟後會看到**主選單「VoiceMod 工具箱」**，每個功能會開成**獨立視窗**，可同時開多個：

- **音效板**：點「＋」格子載入音效（或把檔案拖進去），點一下播放。可設音量、循環、**全域熱鍵**（遊戲中也能按）。
- **文字轉語音**：選語音與語速，打字按「朗讀」就會念出來。

---

## 三、（選用）進階設定

### 要讓聲音進 Discord / 遊戲

需要免費的虛擬麥克風 **VB-Audio Virtual Cable**：

1. 到 https://vb-audio.com/Cable/ 下載 → 解壓 → 對 `VBCABLE_Setup_x64.exe` **右鍵以系統管理員執行** → 裝完**重開機**。
2. 照下表設定：

| 在哪裡 | 設成 |
|--------|------|
| **音效板 →「音訊裝置設定…」** | 輸入＝你的麥克風；輸出＝`CABLE Input` |
| **音效板開關** | 「麥克風直通」開、「監聽(自己聽音效)」開 |
| **Discord → 語音與視訊** | 輸入裝置＝`CABLE Output`；輸出＝你的耳機 |
| **Discord 同頁** | 關掉「自動增益控制」和「噪音抑制 (Krisp)」 |

> 戴耳機避免回授。

### 要載入 MP4 / m4a 當音效

需要 **ffmpeg**：到 https://www.gyan.dev/ffmpeg/builds/ 下載，把 `ffmpeg.exe` 放到程式安裝資料夾裡（捷徑右鍵 →「開啟檔案位置」即可找到）。
（若安裝檔本身已內含 ffmpeg，就不用這步。）

### 要讓文字轉語音念中文

Windows 預設可能只有英文語音。安裝中文語音：
**設定 → 時間與語言 → 語音 → 管理語音 →「新增語音」** → 裝中文，重開程式後就會出現在語音清單。

---

## 四、解除安裝

**設定 → 應用程式 → 已安裝的應用程式** → 找「VoiceMod Toolbox」→ 解除安裝。
（或開始功能表資料夾裡的「Uninstall」捷徑。）

---

## 五、常見問題

| 問題 | 解法 |
|------|------|
| SmartScreen 擋住 | 「其他資訊」→「仍要執行」（未簽章，正常）。 |
| 防毒誤報 | 全域熱鍵用鍵盤掛鉤，加入信任即可（不記錄輸入）。 |
| Discord 對方聽不到 | 確認 Discord 輸入＝`CABLE Output`，並關掉 AGC / Krisp。 |
| 自己聽到自己回音 | 關掉 Windows 裡 `CABLE Output` 的「Listen 接聽」；改用 app 的「監聽」開關。 |
| MP4 載入失敗 | 需要 `ffmpeg.exe`（見上方）。 |
| TTS 念不出中文 | 安裝中文語音（見上方）。 |
